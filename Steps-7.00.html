How to create relationship between fact and dimensions table? 7

The Fact and Dimension tables are connected through the primary and foreign key relationship.  The primary key are located on the dimension tables, while the foreign keys are located in the fact table.  This session show how  to create the relationship between fact and dimension tables.

To create, click the plus sign (+) of the fact  table  to locate the KEY folder.

Next, right click on the Key and select New Foreign key 

Under tables and Columns specific, Click the dots

As shown on the screenshot below follow the steps to create the foreign key relationship.


 1 indicates  - Selecting from the dimension table 

2-indicates - selecting the primary key of a dimension table 

3-indicates-   foreign key of the fact table 


Lastly, Click the Close button to close as shown below.


To add the remaining foreign key relationships, locate the script from the code file and execute.
